# SafeLog Password Validator 🔐

A Core Java project that validates employee passwords based on company security policies.

## Features

✔ Minimum 8 characters  
✔ At least one uppercase letter  
✔ At least one digit  
✔ Specific error messages  
✔ Retry until valid password entered  

## Technologies Used

- Java
- Scanner Class
- Loops
- String Handling

## How It Works

The user enters a password.  
The program checks:

- Length
- Uppercase letters
- Digits

If invalid, it tells the user why and asks again.

## Example Output

Enter your password: hello

Password is too short  
Password must contain uppercase letter  
Password must contain digit

Please try again.

Enter your password: Hello123

Password accepted. Strong password!
